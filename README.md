# River City Martial Arts Academy Sample Asset Set

This package contains a simplified sample asset set for a fictitious martial arts studio website.

## Included
- `build-brief.md` for website build testing
- `copy/site-copy.md` for main page copy
- `copy/more-copy.md` for supporting copy
- `data/business.json` for core business details
- `data/links.json` for sample public profile links
- `data/ctas.json` for key CTA links
- `data/programs.json` for structured programs and pricing data
- `data/team.json` for instructor information
- `docs/programs.pdf` for a sample downloadable programs guide
- `prompts/nano-prompt.md` for one combined Nano Banana prompt
- `img/README.md` describing the expected image outputs
- `logo/README.md` describing the expected logo output

## Not included
- No final logo file
- No final image files
- No brand color file

Use Nano Banana to generate the logo and photo assets from the prompt in the `prompts` folder.
